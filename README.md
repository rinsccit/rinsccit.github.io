# My Personal Portfolio

This repository contains my personal portfolio website and online curriculum vitae. The project is built with React, HyperText Markup Language, Cascading Style Sheets and JavaScript.

## Instructions

### Setup

```shell
git clone https://github.com/rinsccit/my-personal-portfolio.git
cd my-personal-portfolio
```

Install dependencies:

```shell
npm install
```

### Run the Development Server

Start the local development server:

```shell
npm start
```

### Useful Commands

Run static code analysis:

```shell
npm run lint
```

Create a production build:

```shell
npm run build
```

Run tests:

```shell
npm test
```

Deploy the build folder (GitHub Pages):

```shell
npm run deploy
```

## Customisation Notes

- Edit personal and section content in src/portfolio.js.
- Replace or add images in public/images.
- Add documents (for example, your curriculum vitae in Portable Document Format) in public/documents.
